# REPortal - Biblioteca de Juegos 🎮

Un portal centralizado para gestionar y vincular tus bibliotecas de videojuegos de múltiples plataformas en un solo lugar.

## 📋 Descripción

**REPortal** es un proyecto enfocado en crear un portal de videojuegos donde puedas:

- **Gestionar tus bibliotecas**: Vincula y organiza tu biblioteca de juegos de diferentes plataformas (Steam, PlayStation, Xbox, Nintendo, Epic Games, etc.)
- **Vista unificada**: Accede a toda tu colección de juegos desde un único apartado
- **Noticias y actualizaciones**: Mantente informado sobre los últimos lanzamientos, actualizaciones y noticias de la industria gaming
- **Curiosidades**: Descubre datos interesantes y trivia sobre tus juegos favoritos

## 🛠️ Tecnologías

Este proyecto utiliza:
- **SCSS** (54.7%) - Estilos y diseño responsivo
- **TypeScript** (27.4%) - Lógica de la aplicación
- **HTML** (17.9%) - Estructura del portal

## 🎯 Objetivo

Proporcionar a los jugadores una herramienta integral para tener un control total de sus bibliotecas de juegos y mantenerse al día con el mundo gaming, eliminando la necesidad de navegar entre múltiples plataformas.

---

**Nota**: Este es un fork de un proyecto existente, mejorado y adaptado para ofrecerte una experiencia superior en la gestión de tu biblioteca personal de videojuegos.
